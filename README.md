📌 Repository Name
vue-starter

📝 Description
A Vue.js restaurant management app featuring CRUD operations with sign-up and login functionality. This project is a great way to learn about Vue Router, mounted lifecycle hooks, async/await, axios, JSON server, and local storage for user session management.

🚀 Features:
✅ CRUD Operations – Create, Read, Update, and Delete restaurants
✅ Sign-Up & Login – User authentication with local storage
✅ Vue Router – Implement smooth navigation between pages
✅ Axios – Fetch restaurant data from a fake JSON server
✅ Mounted Hook – Load data when components mount
✅ Async/Await – Efficient handling of API requests
